# Tabuada - HTML e CSS embutidos no Python

Este repositório demonstra uma forma simples de manter um HTML com CSS básico dentro de um arquivo Python, gerando uma página que exibe o código de uma tabuada (for) e a sequência de pares (while) dentro de uma caixa com borda e box-sizing: border-box.

Link do projeto (Render):
https://tabuada-html-css-embutido-no-python.onrender.com

## O que tem no projeto
- tabuada.py: contém o HTML e o CSS embutidos em uma string e a lógica para exibir o conteúdo na página.

## Objetivo
Apresentar um exemplo didático de como organizar e renderizar uma página HTML simples a partir de um arquivo Python, mantendo a estilização mínima e fácil de entender.
